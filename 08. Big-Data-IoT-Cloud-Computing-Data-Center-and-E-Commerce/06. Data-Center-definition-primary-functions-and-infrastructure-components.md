# Data Center — Definition, Primary Functions & Infrastructure Components

Foundational overview of what a data center is and the core elements that make it operate.

- **Definition** — a centralized facility housing computing, storage, and networking equipment for IT operations
- **Primary functions**: data processing, storage, backup, disaster recovery, hosting applications/services
- **Infrastructure components**:
  - **Compute** — servers (rack, blade, tower)
  - **Storage** — SAN, NAS, DAS, object storage
  - **Networking** — switches, routers, firewalls, load balancers
  - **Power** — UPS, generators, PDU
  - **Cooling** — CRAC units, chillers, liquid cooling
  - **Physical security** — biometric access, surveillance, mantraps
