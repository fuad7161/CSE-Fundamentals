# Server Virtualization in Data Centers

How virtualization technology improves resource utilization and operational efficiency.

- **Definition** — abstracting physical hardware to run multiple virtual machines (VMs) on a single physical server
- **Hypervisor** — software layer that manages VMs (Type 1: bare-metal, Type 2: hosted)
  - Examples: VMware ESXi, Microsoft Hyper-V, KVM, Xen
- **Benefits**: server consolidation, reduced power/cooling costs, faster provisioning, isolation
- **Key concepts**: VM, virtual CPU/memory/storage, live migration, snapshots
- **Containers vs. VMs** — lightweight alternative using OS-level virtualization (e.g., Docker, Kubernetes)
