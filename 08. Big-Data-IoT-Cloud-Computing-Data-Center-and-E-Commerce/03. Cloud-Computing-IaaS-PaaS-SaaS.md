# Cloud Computing — IaaS, PaaS & SaaS

Fundamentals of cloud service models and their delivery characteristics.

- **Cloud Computing** — on-demand delivery of computing resources over the internet
- **Infrastructure as a Service (IaaS)** — virtualized compute, storage, networking (e.g., AWS EC2, Google Compute Engine)
- **Platform as a Service (PaaS)** — managed runtime environment for application development (e.g., Heroku, Google App Engine)
- **Software as a Service (SaaS)** — ready-to-use applications accessed via browser (e.g., Gmail, Salesforce)
- **Deployment models**: public, private, hybrid, community cloud
- **Key characteristics**: on-demand self-service, broad network access, resource pooling, rapid elasticity, measured service
