# UPS — Purpose in a Data Center

Role of Uninterruptible Power Supply in ensuring continuous data center operation.

- **Definition** — device that provides emergency backup power when mains power fails
- **Primary purpose**: bridge the gap between power failure and generator startup (ride-through)
- **UPS types**:
  - **Standby (Offline)** — switches to battery only during fault
  - **Line-Interactive** — regulates voltage with auto-transformer
  - **Double-Conversion (Online)** — continuously regenerates clean AC power; highest protection
- **Key metrics**: runtime (minutes at full load), transfer time, efficiency, battery type (VRLA, Lithium-ion)
- **N+1 redundancy** — critical for high-availability data centers
