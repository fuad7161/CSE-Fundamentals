# Number Systems — Binary, Octal, Hex Conversions

Foundational number systems used in digital electronics and computing. Covers:

- **Binary (base-2)** — bits, bytes, binary arithmetic
- **Octal (base-8)** — grouping bits in threes
- **Hexadecimal (base-16)** — grouping bits in fours, memory addressing
- Conversion methods between binary, octal, hex, and decimal
- Applications: memory addresses, color codes, machine-level representation
