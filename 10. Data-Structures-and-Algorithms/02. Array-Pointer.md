# Array and Pointer

Covers the fundamental concepts of arrays and pointers, their relationship, and common use cases.

- Array declaration, initialization, and memory layout
- One-dimensional and multi-dimensional arrays
- Pointer basics: declaration, dereferencing, and pointer arithmetic
- Relationship between arrays and pointers
- Dynamic memory allocation for arrays
- Common operations: traversal, insertion, deletion, and searching
- Pointer to pointer, function pointers, and pointer arrays
