# Common Algorithms — Line Drawing, Clipping, Transformations

Core algorithms used to render and manipulate 2D and 3D geometry on raster displays.

- **Line Drawing Algorithms** — DDA algorithm, Bresenham's line algorithm (integer arithmetic)
- **Circle Drawing** — Midpoint circle algorithm (Bresenham's circle)
- **Clipping** — Cohen–Sutherland line clipping, Sutherland–Hodgman polygon clipping
- **2D Transformations** — translation, rotation, scaling, reflection, shearing (homogeneous coordinates)
- **3D Transformations** — rotation about axes, perspective vs orthographic projection
- **Compositing** — matrix concatenation for compound transformations
