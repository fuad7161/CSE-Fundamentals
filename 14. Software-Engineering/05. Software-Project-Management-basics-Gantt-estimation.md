# Software Project Management basics — Gantt, estimation

Foundational project management concepts for software engineering projects.

- **Project management triangle**: scope, time, cost (quality at the center)
- **Gantt charts**: visual timeline of tasks, dependencies, and milestones
- **Estimation techniques**:
  - Expert judgment
  - Analogous estimation (based on similar past projects)
  - Parametric estimation (using historical data / metrics)
  - Three-point estimation (optimistic, pessimistic, most likely)
  - COCOMO model (Constructive Cost Model)
- Work breakdown structure (WBS)
- Risk management and resource allocation basics
- Tracking progress: burn-down charts, earned value management
