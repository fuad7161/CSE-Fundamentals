# Arrays & Functions

Understanding how to store collections of data and how to define reusable blocks of code.

- **Arrays** — contiguous memory storage for elements of the same data type
  - One-dimensional arrays, multi-dimensional arrays, array initialization
  - Indexing, traversal, common operations (search, sort, insert, delete)
- **Functions** — named, reusable block of code that performs a specific task
  - Function declaration (prototype) and definition
  - Parameters: pass-by-value, pass-by-reference / pass-by-address
  - Return types and the `void` type
  - Scope and lifetime of variables (local, global, static)
- **Recursion** — function that calls itself; base case and recursive case
- **Standard library functions** — e.g., `printf/scanf` (C), `cout/cin` (C++), `System.out` (Java), `print/input` (Python)
