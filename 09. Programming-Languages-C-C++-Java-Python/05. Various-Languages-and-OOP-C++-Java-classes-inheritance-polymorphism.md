# Various Languages & OOP (C++/Java) — Classes, Inheritance, Polymorphism

Comparison of programming languages and the core pillars of object-oriented programming.

- **Language overview**:
  - **C** — procedural, low-level, manual memory management
  - **C++** — multi-paradigm (procedural + OOP), compiled, high performance
  - **Java** — OOP-first, compiled to bytecode (JVM), garbage-collected
  - **Python** — interpreted, dynamically typed, multi-paradigm
- **Object-Oriented Programming (OOP)**:
  - **Classes & Objects** — blueprints and instances
  - **Encapsulation** — bundling data with methods; access specifiers (public, private, protected)
  - **Inheritance** — deriving new classes from existing ones (`extends` in Java, `:` in C++)
    - Single, multilevel, hierarchical, multiple (C++ only), hybrid
  - **Polymorphism** — same interface, different implementations
    - Compile-time: function overloading, operator overloading (C++)
    - Runtime: virtual functions (C++), method overriding (Java)
  - **Abstraction** — hiding complex implementation (abstract classes, interfaces)
