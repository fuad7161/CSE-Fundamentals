# ALU and Control Unit functions

Core roles and functions of the Arithmetic Logic Unit and Control Unit in a CPU.

- **ALU (Arithmetic Logic Unit)**:
  - Arithmetic operations: addition, subtraction, increment, decrement
  - Logical operations: AND, OR, XOR, NOT
  - Shift and rotate operations
  - Flag generation (zero, carry, overflow, sign)
- **Control Unit**:
  - Instruction fetch, decode, and sequencing
  - Generation of control signals (register read/write, ALU op, memory R/W)
  - Hardwired vs. microprogrammed control
  - Timing and synchronization of datapath
