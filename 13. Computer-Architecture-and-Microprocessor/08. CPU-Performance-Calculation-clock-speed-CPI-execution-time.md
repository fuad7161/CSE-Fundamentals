# CPU Performance Calculation — clock speed, CPI, execution time

How to measure and calculate CPU performance using key metrics.

- **Clock speed** (frequency): cycles per second (Hz)
- **CPI (Cycles Per Instruction)**: average cycles required per instruction
- **Instruction count**: total number of instructions executed
- **Execution time** formula:  
  `Time = (Instruction Count × CPI) / Clock Rate`
  or  
  `Time = (Instruction Count × CPI) × Clock Cycle Time`
- **MIPS** (Millions of Instructions Per Second)
- Amdahl's Law and the impact of improving a single component
- Worked example: calculating execution time given clock rate, CPI, and instruction count
