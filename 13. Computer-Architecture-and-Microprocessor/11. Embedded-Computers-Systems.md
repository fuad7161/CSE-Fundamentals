# Embedded Computers & Systems

Introduction to embedded systems — specialized computing systems dedicated to specific tasks.

- Definition of embedded systems vs. general-purpose computers
- Common components: microcontroller/microprocessor, memory, I/O peripherals
- Characteristics: real-time constraints, low power, cost-sensitive, dedicated function
- Examples: automotive ECUs, smart appliances, IoT devices, medical devices
- Microcontrollers vs. microprocessors (integrated peripherals, on-chip memory)
- Real-time operating systems (RTOS) and bare-metal programming
- Development tools: cross-compilers, debuggers, simulators
