# Computer Files

Overview of computer file systems, file types, and organization.

- File as a named collection of data on a storage medium
- File attributes: name, extension, size, timestamps, permissions
- Common file types: text (.txt), executables (.exe), images, archives
- File systems: FAT32, NTFS, ext4, APFS
- Directory/ folder hierarchy and path notation (absolute vs. relative)
- File operations: create, open, read, write, delete
- File permissions and access control
