# Cryptography — Symmetric vs. Asymmetric & Hashing

Techniques for secure communication and data integrity. Covers:

- **Symmetric encryption** — same key for encryption/decryption (AES, DES, 3DES)
- **Asymmetric encryption** — public/private key pairs (RSA, ECC, Diffie-Hellman)
- **Hashing** — one-way functions for integrity verification (SHA-256, MD5, bcrypt)
- Digital signatures and certificates
- Trade-offs: speed vs. key management complexity
