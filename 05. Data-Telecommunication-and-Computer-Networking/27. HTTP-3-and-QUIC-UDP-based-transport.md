# HTTP/3 and QUIC — UDP-Based Transport

Next-generation HTTP protocol built on QUIC instead of TCP. Covers:

- **QUIC** — transport protocol developed by Google, runs over UDP
- **HTTP/3** — HTTP semantics mapped onto QUIC
- Key advantages: reduced connection establishment latency, improved multiplexing, built-in encryption
- Better performance on lossy networks (no TCP head-of-line blocking)
- Migration path and current adoption status
