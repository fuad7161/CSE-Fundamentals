# Network Devices — Router, Switch, Hub, Bridge, Gateway

Hardware components that connect and manage network traffic. Covers:

- **Hub** — broadcasts data to all ports (layer 1)
- **Bridge** — segments network by MAC addresses (layer 2)
- **Switch** — intelligent forwarding based on MAC addresses (layer 2/3)
- **Router** — forwards packets between networks (layer 3)
- **Gateway** — translates between different protocols/networks
- Firewall, modem, access point, and repeater roles
