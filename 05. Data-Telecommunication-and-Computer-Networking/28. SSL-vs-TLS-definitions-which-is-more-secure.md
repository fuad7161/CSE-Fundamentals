# SSL vs. TLS — Definitions, Which Is More Secure

Cryptographic protocols for securing Internet communications. Covers:

- **SSL (Secure Sockets Layer)** — original protocol (v2/v3), now deprecated
- **TLS (Transport Layer Security)** — successor to SSL (v1.0–v1.3)
- Key improvements in TLS: stronger cipher suites, Perfect Forward Secrecy, 1-RTT handshake
- TLS 1.3 removes obsolete features, reduces handshake latency
- **TLS is the secure, modern standard** — SSL should not be used
