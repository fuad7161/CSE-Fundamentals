# Others — Miscellaneous Protocol Trivia

Assorted facts and notable details about network protocols. Covers:

- **FTP** — two-port design (21 control, 20 data); active vs. passive mode
- **Telnet vs. SSH** — plaintext vs. encrypted remote shell
- **SNMP** — network management and monitoring (MIB, OID)
- **ARP** — resolves IP to MAC addresses; spoofing risk
- **ICMP** — error reporting (ping, traceroute)
- **NTP** — time synchronization across networks
- **SMB/NFS** — file sharing protocols
- **RIP, OSPF, BGP** — routing protocol overview
