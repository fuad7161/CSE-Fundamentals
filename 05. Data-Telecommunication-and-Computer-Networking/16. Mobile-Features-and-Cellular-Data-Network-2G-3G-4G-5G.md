# Mobile Features and Cellular Data Network — 2G to 5G

Evolution of mobile communication standards. Covers:

- **2G** — digital voice, SMS, GSM/CDMA
- **3G** — mobile data, video calls, UMTS/HSPA
- **4G/LTE** — high-speed broadband, all-IP network, VoLTE
- **5G** — ultra-low latency, mmWave, massive MIMO, network slicing
- Key metrics: data rate, latency, frequency bands, and use cases per generation
