# Machine Learning — Supervised, Unsupervised & Reinforcement Basics

Overview of the three core paradigms of machine learning and how they differ.

- **Supervised Learning** — model trained on labeled data (input-output pairs)
  - Regression (predict continuous values) & Classification (predict categories)
  - Examples: Linear Regression, Decision Trees, SVM, Neural Networks
- **Unsupervised Learning** — model finds patterns in unlabeled data
  - Clustering (group similar items) & Dimensionality Reduction
  - Examples: K-Means, DBSCAN, PCA, Autoencoders
- **Reinforcement Learning** — agent learns by interacting with environment through rewards/penalties
  - Key concepts: state, action, reward, policy, value function
  - Examples: Q-Learning, Deep Q-Networks, Policy Gradients
