# Tree and Graph

Covers hierarchical (tree) and networked (graph) data structures, their representations, and traversal algorithms.

- **Trees:** binary tree, binary search tree (BST), AVL tree, heap, and B-tree
- Tree traversal: inorder, preorder, postorder, and level-order
- **Graphs:** directed vs. undirected, weighted vs. unweighted
- Graph representations: adjacency matrix and adjacency list
- Graph traversal: BFS (Breadth-First Search) and DFS (Depth-First Search)
- Applications: shortest path (Dijkstra's), minimum spanning tree (Kruskal's, Prim's)
- Complexity analysis of tree and graph operations
