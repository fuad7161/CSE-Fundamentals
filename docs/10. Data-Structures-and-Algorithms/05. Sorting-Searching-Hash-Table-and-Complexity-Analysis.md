# Sorting, Searching, Hash Table, and Complexity Analysis

Covers classic sorting and searching algorithms, hash tables, and comparative complexity analysis.

- **Sorting algorithms:** bubble sort, selection sort, insertion sort, merge sort, quick sort, heap sort
- **Searching algorithms:** linear search and binary search
- **Hash table:** hash functions, collision resolution (chaining, open addressing), load factor
- Time and space complexity comparisons of sorting and searching algorithms
- Stability, in-place vs. out-of-place sorting
- Practical considerations for selecting the right algorithm
