# Linked List, Stack, and Queue

Explores linear data structures — linked lists, stacks, and queues — their implementations and applications.

- **Linked List:** singly, doubly, and circular linked lists; insertion, deletion, and traversal
- **Stack:** LIFO principle, push/pop operations, array and linked-list implementations
- **Queue:** FIFO principle, enqueue/dequeue operations, circular queue, and priority queue
- Applications: expression evaluation (stack), scheduling (queue), memory management
- Time complexity analysis for each operation
