# Digital Logic Gates & K-Map

Building blocks of digital circuits and minimization technique. Covers:

- **Logic gates** — AND, OR, NOT, NAND, NOR, XOR, XNOR (symbols, truth tables)
- Universal gates (NAND, NOR) — can implement any Boolean function
- **Karnaugh Map (K-Map)** — graphical method for simplifying Boolean expressions
- 2, 3, and 4-variable K-Maps (grouping, don't-care conditions)
- Comparison of K-Map vs. algebraic simplification
