# Operators & Control Flow Loops

How operators manipulate data and how control structures direct program execution.

- **Operators**:
  - Arithmetic: `+`, `-`, `*`, `/`, `%`
  - Relational: `==`, `!=`, `<`, `>`, `<=`, `>=`
  - Logical: `&&`, `||`, `!`
  - Bitwise: `&`, `|`, `^`, `~`, `<<`, `>>`
  - Assignment: `=`, `+=`, `-=`, etc.
- **Conditional statements**: `if`, `else if`, `else`, `switch-case`
- **Looping constructs**:
  - `for` loop — known iteration count
  - `while` loop — condition-based iteration
  - `do-while` loop — executes body at least once
- **Jump statements**: `break`, `continue`, `return`, `goto`
- **Common patterns**: nested loops, loop control variables, infinite loops
