# Computer Programming Fundamentals

Core concepts common to all programming languages and the software development process.

- **What is programming?** — writing instructions for a computer to execute
- **Programming paradigms**: procedural, object-oriented, functional, logic
- **Basic building blocks**: variables, data types, operators, control flow, functions
- **Program lifecycle**: source code → compilation/interpreter → execution
- **Development tools**: editors, compilers, interpreters, debuggers, version control (Git)
- **Problem-solving approach**: understand problem → design algorithm → code → test → debug
