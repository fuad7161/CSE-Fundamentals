# Output Prediction — MCQ Practice (Trace Code, Predict Output)

Practice-focused section for solving code tracing and output prediction problems.

- **What to expect** — multiple-choice questions testing ability to trace program execution
- **Common question patterns**:
  - Operator precedence and associativity
  - Loop iterations and boundary conditions
  - Function call stack and recursion depth
  - Pointer arithmetic and memory access (C/C++)
  - Object construction/destruction order (C++/Java)
  - Static variable behavior across function calls
  - String manipulation edge cases
  - Type casting and implicit conversions
- **Tips**: dry-run with a trace table, pay attention to side effects, watch for integer overflow, off-by-one errors
