# Strings, Pointers & File Handling

Text manipulation, memory management through pointers, and persistent data I/O.

- **Strings** — sequences of characters
  - C-style strings (`char[]` null-terminated) std::string (C++), String class (Java), str (Python)
  - Common operations: concatenation, comparison, substring, searching
- **Pointers** — variables that store memory addresses
  - Pointer arithmetic, dereferencing, pointers to arrays, function pointers
  - Dynamic memory allocation: `malloc`/`free` (C), `new`/`delete` (C++)
  - Null pointers, dangling pointers, memory leaks
- **File handling** — reading from and writing to files
  - C: `fopen`, `fclose`, `fprintf`, `fscanf`, `fread`, `fwrite`
  - C++: `fstream`, `ifstream`, `ofstream`
  - Java: `FileReader`, `BufferedReader`, `FileWriter`
  - Python: `open()`, `.read()`, `.write()`, context manager (`with`)
- **Error handling**: checking file open status, EOF detection, exception handling
