# C Programming — Dedicated Practice Set

Targeted practice problems covering core C programming topics.

- **Focus areas**:
  - Basic I/O, data types, and format specifiers
  - Control flow — conditional statements, loops
  - Arrays — 1D/2D manipulation, matrix operations
  - Functions — recursion, passing arrays to functions
  - Pointers — pointer arithmetic, pointer-to-pointer, dynamic memory (`malloc`/`calloc`/`realloc`/`free`)
  - Strings — standard library functions (`strlen`, `strcpy`, `strcmp`, `strcat`)
  - Structures and unions
  - File I/O — reading/writing text and binary files
  - Bitwise operations and bit manipulation
- **Problem types**: predict output, find errors, complete the code, write short programs
- **Recommended approach**: compile and test with `gcc`, use a debugger (`gdb`) for tracing
