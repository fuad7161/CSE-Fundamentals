# Internet of Things (IoT)

Overview of the network of interconnected physical devices that collect and exchange data.

- **Definition** — everyday objects embedded with sensors, software, and connectivity
- **IoT architecture**: perception layer, network layer, middleware layer, application layer
- **Communication protocols**: MQTT, CoAP, HTTP/2, Bluetooth, Zigbee, LoRaWAN
- **Key components**: sensors, actuators, gateways, cloud platforms
- **Applications**: smart homes, wearables, industrial IoT (IIoT), smart cities, healthcare
- **Challenges**: security, interoperability, data privacy, power management
