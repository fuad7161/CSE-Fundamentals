# Big Data — 3V/5V & Hadoop Basics

Fundamental characteristics of big data and the ecosystem built to process it.

- **3Vs of Big Data**: Volume (scale), Velocity (speed), Variety (diversity)
- **Extended 5Vs**: adding Veracity (data quality) and Value (business insight)
- **Hadoop** — open-source framework for distributed storage and processing
  - **HDFS** (Hadoop Distributed File System) — fault-tolerant storage across clusters
  - **MapReduce** — parallel processing model (map → shuffle → reduce)
  - **YARN** — cluster resource management and job scheduling
- **Hadoop ecosystem tools**: Hive, Pig, HBase, Spark, Flume, Sqoop
