## ৩. পূর্ববর্তী পরীক্ষায় বারবার আসা প্রশ্নসমূহ (Frequently Asked Questions)

নিচের প্রশ্নগুলো বিগত পরীক্ষার প্যাটার্ন অনুসারে বার বার এসেছে — এগুলো ভালোভাবে অনুশীলন করুন:

31. Explain DNS, DHCP, Active Directory briefly.
32. Type-1 ও Type-2 Hypervisor ব্যাখ্যা করুন।
33. Briefly describe primary key, foreign key and index in a relational database and their relationship. Do you think database indexing always makes applications faster? Explain your answer.
34. How to solve drive failure in RAID?
35. Write down the computer architecture characteristics. Write down the stages of the DLX pipeline.
36. Write down four common rules of Assembly language. Explain different types of hazards.
37. Given an instruction in an 8086-based Assembly language program, `MOV BL, AL`. Convert this assembly language instruction into machine code in HEX.
38. Computer A has a 3.2 GHz processing speed and 2.0 clock cycles per instruction (CPI) for a program; Computer B has a 2.4 GHz processing speed with 1.2 CPI for the same program. Which computer runs faster, and by how much?
39. Describe the addressing modes of the 8086 microprocessor.
40. Explain the functions of the ALU and Control Unit of a computer.
41. What is HTTP, and what is its main purpose in web communication?
42. Write two major differences between HTTP/1.1 and HTTP/2.
43. Why is HTTP/2 faster than HTTP/1.1?
44. What protocol does HTTP/3 use instead of TCP, and why?
45. Define SSL and TLS. Which one is more secure today?
46. What is the purpose of an SSL/TLS certificate?
47. What is the default port number of HTTP and HTTPS? What is a digital certificate, and who issues it?
48. What is a data center, and what are its primary functions? Explain the main components of data center infrastructure.
49. What is the difference between a traditional data center and a cloud data center? What is server virtualization, and why is it important in data centers?
50. What is the purpose of a UPS in a data center? Explain the difference between Tier I, Tier II, Tier III, and Tier IV data centers. What is cooling management in a data center, and why is it necessary?
