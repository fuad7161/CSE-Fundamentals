# SDLC Models — Waterfall, Agile, Spiral, Iterative

Overview of the major Software Development Life Cycle models and their trade-offs.

- **Waterfall**: linear sequential phases (requirements → design → implementation → verification → maintenance)
- **Agile**: iterative, incremental delivery; Scrum and Kanban frameworks
- **Spiral**: risk-driven iterative model combining prototyping and waterfall
- **Iterative**: repeated cycles of development with feedback loops
- Comparison of strengths, weaknesses, and appropriate use cases for each model
- Choosing the right model based on project size, risk, and requirements stability
