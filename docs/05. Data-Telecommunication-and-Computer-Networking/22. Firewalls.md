# Firewalls

Network security system that monitors and controls incoming/outgoing traffic. Covers:

- Types: packet-filtering, stateful inspection, proxy, next-generation firewall (NGFW)
- Rule-based access control (allow/deny by IP, port, protocol)
- Firewall placement in network architecture
- Comparison of hardware vs. software firewalls
- Limitations: cannot prevent internal threats or all malware
