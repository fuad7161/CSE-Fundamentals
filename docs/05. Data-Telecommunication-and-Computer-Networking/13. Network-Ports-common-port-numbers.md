# Network Ports — Common Port Numbers

Well-known ports used by network protocols and services. Covers:

- **20/21** — FTP (data/control)
- **22** — SSH
- **23** — Telnet
- **25** — SMTP
- **53** — DNS
- **80** — HTTP
- **110** — POP3
- **143** — IMAP
- **443** — HTTPS
- **3389** — RDP
- Understanding port ranges: well-known (0–1023), registered (1024–49151), dynamic (49152–65535)
