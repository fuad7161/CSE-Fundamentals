# Computer Network Types — LAN, MAN, WAN, PAN

Classification of networks by geographic scope. Covers:

- **PAN (Personal Area Network)** — few meters, personal devices
- **LAN (Local Area Network)** — building or campus, high speed
- **MAN (Metropolitan Area Network)** — city-wide coverage
- **WAN (Wide Area Network)** — country/continent scale, slower
- Comparisons by size, speed, latency, and ownership
