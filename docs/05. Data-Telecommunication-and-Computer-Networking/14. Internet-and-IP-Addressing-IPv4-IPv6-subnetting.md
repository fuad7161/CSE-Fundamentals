# Internet and IP Addressing — IPv4, IPv6, Subnetting

Fundamentals of IP addressing and network division. Covers:

- **IPv4** — 32-bit addressing, classes (A, B, C, D, E), public vs. private addresses
- **IPv6** — 128-bit addressing, hexadecimal notation, benefits over IPv4
- **Subnetting** — dividing a network using subnet masks, CIDR notation
- VLSM (Variable Length Subnet Masking) and route aggregation
- NAT (Network Address Translation) and its role
