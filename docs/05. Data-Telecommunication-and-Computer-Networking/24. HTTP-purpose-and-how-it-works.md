# HTTP — Purpose and How It Works

The Hypertext Transfer Protocol powers communication on the World Wide Web. Covers:

- Client-server model: browser sends request, server returns response
- HTTP methods: GET, POST, PUT, DELETE, PATCH, HEAD, OPTIONS
- Request/response structure (headers, body, status codes)
- Stateless nature and how sessions maintain state (cookies, tokens)
- Common status codes: 200, 301, 400, 401, 403, 404, 500, 502, 503
