# Database Keys and Indexing

Explains the various types of keys in relational databases and the role of indexing.

- **Keys:** super key, candidate key, primary key, foreign key, composite key, alternate key
- Entity integrity and referential integrity constraints
- **Indexing:** clustered vs. non-clustered indexes
- How indexes improve query performance
- B-tree and hash-based indexes
- Best practices for key selection and index design
