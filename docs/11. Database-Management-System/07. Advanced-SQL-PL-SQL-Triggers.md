# Advanced SQL: PL/SQL and Triggers

Covers procedural extensions to SQL and database triggers for automation and business logic.

- **PL/SQL:** blocks, variables, control structures (IF, LOOP), cursors, exceptions
- Stored procedures and functions: creation, parameters, and return values
- **Triggers:** BEFORE, AFTER, INSTEAD OF; row-level and statement-level triggers
- Use cases: auditing, validation, derived data, logging
- Packages in PL/SQL
- Performance and maintainability considerations
- Comparison with other procedural SQL extensions (T-SQL, pgSQL)
