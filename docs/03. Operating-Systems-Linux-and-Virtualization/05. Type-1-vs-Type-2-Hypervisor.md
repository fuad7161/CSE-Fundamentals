# Type 1 vs. Type 2 Hypervisor

Compares the two categories of hypervisors — bare-metal (Type 1) and hosted (Type 2) — and their typical deployment scenarios.

- Type 1: runs directly on hardware (VMware ESXi, Microsoft Hyper-V, Xen)
- Type 2: runs on top of a host OS (VirtualBox, VMware Workstation)
- Performance, security, and resource overhead differences
- Enterprise vs. desktop use cases
