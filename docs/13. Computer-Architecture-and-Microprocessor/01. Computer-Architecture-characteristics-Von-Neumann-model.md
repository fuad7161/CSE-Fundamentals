# Computer Architecture characteristics & Von Neumann model

Brief overview of computer architecture fundamentals and the Von Neumann architecture.

- Definition of computer architecture (ISA, microarchitecture, hardware organization)
- Von Neumann model: stored-program concept, single memory for instructions and data
- Key components: CPU (ALU + Control Unit), memory, input/output
- Von Neumann bottleneck — the limitation imposed by shared bus between CPU and memory
- Contrast with Harvard architecture (separate instruction and data memories)
