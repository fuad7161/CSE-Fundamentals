# Pipeline Hazards — structural, data, control hazards

Covers the three main types of pipeline hazards and their mitigation.

- **Structural hazards**: resource conflicts (e.g., single memory for instructions and data)
- **Data hazards**: RAW (read-after-write), WAR, WAW dependencies
  - Forwarding/bypassing, stalling, reordering
- **Control hazards**: branch instructions disrupting the pipeline flow
  - Branch prediction, delayed branching, flush/recovery
- Impact on CPI and pipeline performance
