# Assembly Language — common rules & conventions

Fundamentals of assembly language syntax, structure, and programming conventions.

- Instruction format: opcode, operands (register, immediate, memory)
- Common x86/8086 mnemonics: MOV, ADD, SUB, JMP, CMP, INT
- Directives (e.g., DB, DW, ORG, END) and labels
- Register naming and usage conventions
- Comment syntax and program structure (data segment, code segment, stack segment)
