# 8086 Machine Code conversion — MOV BL, AL to HEX

Walkthrough of converting an 8086 assembly instruction into its hexadecimal machine code representation.

- 8086 instruction encoding format (opcode, d bit, w bit, mod, reg, r/m)
- Meaning of `d` (direction) and `w` (word/byte) fields
- ModR/M byte breakdown: mod, reg, r/m fields
- Step-by-step example: `MOV BL, AL`
  - Opcode: 88h (MOV reg/mem8 → reg8) or 8Ah (MOV reg8 → reg/mem8)
  - Determining the correct encoding based on direction bit
  - ModR/M byte for register-to-register transfer
  - Final hex representation
- Practice with other register-to-register MOV variants
