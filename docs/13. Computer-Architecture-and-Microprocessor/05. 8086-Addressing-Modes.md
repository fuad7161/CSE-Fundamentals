# 8086 Addressing Modes

Describes the various addressing modes available in the Intel 8086 microprocessor.

- **Immediate**: operand is a constant in the instruction
- **Register**: operand is in a register
- **Direct**: effective address is given directly in the instruction
- **Register Indirect**: address held in a register (BX, SI, DI)
- **Based**: address = base register (BX or BP) + displacement
- **Indexed**: address = index register (SI or DI) + displacement
- **Based Indexed**: address = base + index + displacement
- String addressing (use of SI/DI with direction flag)
