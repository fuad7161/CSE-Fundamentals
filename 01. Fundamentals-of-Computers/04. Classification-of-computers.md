# Classification of Computers

Computers are classified by size, purpose, and data handling capability.

### 1.4 — Classification by Size
- Microcomputer (PC, laptop, tablet)
- Minicomputer
- Mainframe
- Supercomputer

### 1.5 — Classification by Purpose
- General-purpose computer
- Special-purpose computer

### 1.6 — Analog, Digital, and Hybrid Computers
- Analog: continuous data (voltage, temperature)
- Digital: discrete binary data
- Hybrid: combination of both (e.g., ICU monitors)
