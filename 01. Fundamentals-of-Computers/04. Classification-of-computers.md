# Classification of Computers

Computers are classified by size, purpose, and data handling capability.

### 1.4 — Classification by Size
- Microcomputer (PC, laptop, tablet)
- Minicomputer
- Mainframe
- Supercomputer

### 1.5 — Classification by Purpose
- General-purpose computer
- Special-purpose computer

### 1.6 — Analog, Digital, and Hybrid Computers
- Analog: continuous data (voltage, temperature)
- Digital: discrete binary data
- Hybrid: combination of both (e.g., ICU monitors)

## Supplementary Notes from Master Study Guide

### Classification Based on Operating Principles

| Type | Description | Example |
|---|---|---|
| Analog | Continuous electrical signals | Operational amplifiers |
| Digital | Processes data in 0s and 1s | Desktop computers |
| Hybrid | Combines analog and digital | Hospital heartbeat monitors |

### Classification Based on Size and Capability (Detailed)

| Type | Description | Examples |
|---|---|---|
| Microcomputer | Small, personal use | Desktop, Laptop, PDA |
| Mini Computer | Mid-range, 4-200 users | PDP 11, IBM 8000 series |
| Mainframe | Large, enterprise use | IBM 3000, VAX 8000 |
| Super Computer | Fastest, complex operations | CRAY 3, PARAM |
