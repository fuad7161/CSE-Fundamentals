# Booting and Firmware — BIOS vs. UEFI

Covers the startup sequence of a computer, the role of firmware, and the transition from legacy BIOS to modern UEFI.

### 3.1 — Booting Process (Cold Boot vs Warm Boot)
- Cold boot: starting from powered-off state
- Warm boot: restarting without cutting power (Ctrl+Alt+Del)
- Boot sequence: POST → BIOS/UEFI → Boot loader → OS

### 3.2 — BIOS — Function & Role
- Basic Input/Output System
- Stored on ROM chip on motherboard
- Performs hardware initialization and POST
- Loads the boot loader from MBR

### 3.3 — UEFI vs BIOS — Differences
- UEFI: modern replacement, GUI interface
- BIOS: legacy, text-based, 16-bit mode, MBR (max 2TB)
- UEFI: 32/64-bit mode, GPT (supports disks >2TB), Secure Boot
- UEFI has faster boot times

### 3.4 — POST (Power-On Self Test)
- Checks if hardware components (RAM, CPU, storage) are functioning
- Beep codes indicate specific hardware errors

### 3.5 — CMOS & System Configuration
- CMOS: battery-powered memory storing BIOS settings (date, time, boot order)
- CMOS battery (CR2032) — when it dies, settings reset
- Accessing BIOS/UEFI setup (F2, Del, F10 keys)
