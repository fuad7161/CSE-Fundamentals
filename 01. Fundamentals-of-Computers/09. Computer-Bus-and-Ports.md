# Computer Bus and Ports

How data travels between internal components via buses and how external devices connect through various ports.

## 8.1 — Bus — Definition, Types

A **bus** is a communication system that transfers data between components inside a computer. It consists of electrical wires, connectors, and protocols that define how data is transmitted.

### Three Main Types of Buses

#### Data Bus
- Carries the actual data being transferred between components (CPU, memory, I/O).
- **Bidirectional**: Data can flow both to and from the CPU.
- **Bus width**: The number of parallel wires determines how many bits can be transferred at once (e.g., a 64-bit data bus can transfer 8 bytes per transaction).
- Wider data bus → higher throughput.

#### Address Bus
- Carries memory addresses from the CPU to memory or I/O devices.
- **Unidirectional**: Addresses only flow from the CPU outward (CPU → memory, CPU → I/O).
- The width of the address bus determines the maximum addressable memory. For an n-bit address bus, the CPU can address 2ⁿ memory locations.
  - Example: 32-bit address bus → 2³² = 4 GB addressable space.
  - Example: 64-bit address bus → 2⁶⁴ = 16 EB addressable space.

#### Control Bus
- Carries control signals such as:
  - **Read / Write** — tells memory whether to read or write data.
  - **Interrupt (IRQ)** — signals from devices requesting CPU attention.
  - **Clock** — synchronizes operations across components.
  - **Bus Request / Grant** — for arbitration when multiple devices want to use the bus.
- Typically bidirectional, carrying signals in both directions.

### Bus Width and Performance
- A wider bus moves more data per cycle: a 32-bit bus at 100 MHz = 32 × 100 × 10⁶ = 400 MB/s.
- Modern CPUs use **64-bit** data buses and **multiplexed** address/data buses to optimize pin usage.

## 8.2 — System Bus vs Expansion Bus

### System Bus
- Also called the **front-side bus (FSB)** or **memory bus**.
- Connects the **CPU to main memory (RAM)** and the **memory controller**.
- Extremely high speed — operates at or near the CPU's clock speed.
- In modern architectures (Intel Core, AMD Ryzen), the memory controller is integrated into the CPU, and the system bus connects directly to RAM via **DDR channels**.

### Expansion Bus
- Connects the CPU to **peripheral devices** (graphics cards, storage controllers, network adapters).
- Slower than the system bus but allows for modular expansion.
- Key expansion bus standards:

| Bus | Speed | Use |
|-----|-------|-----|
| **PCIe (PCI Express)** | Up to ~31.5 GB/s (x16 Gen 4) | Graphics cards, SSDs, network cards |
| **SATA** | 6 Gbps (SATA III) | Hard drives, SSDs, optical drives |
| **USB** | Up to 40 Gbps (USB4) | Keyboards, mice, printers, external storage |
| **Thunderbolt** | Up to 40 Gbps | High-speed peripherals, displays |
| **M.2 (NVMe)** | Up to 8 GB/s (PCIe Gen 5 x4) | SSDs, Wi-Fi cards |

### Historical Note
Older expansion buses include **ISA** (8/16-bit, slow), **PCI** (32/64-bit, 133 MB/s), and **AGP** (dedicated for graphics, now replaced by PCIe).

## 8.3 — Serial Port vs Parallel Port

### Serial Port
- Transmits data **one bit at a time** over a single wire (or differential pair).
- Slower per transaction, but can operate at higher frequencies with less signal degradation.
- Advantages: longer cable distances, fewer wires, cheaper cables, simpler connector.
- Examples:
  - **RS-232**: Traditional serial port (9-pin D-sub), used for modems, mice, industrial equipment. Speeds up to ~115.2 kbps.
  - **USB**: Modern serial standard. Much faster, hot-swappable.
  - **SATA**: Serial replacement for parallel ATA (PATA).
  - **PCIe**: Serial replacement for PCI/AGP.

### Parallel Port
- Transmits **multiple bits simultaneously** over several parallel wires (typically 8 bits at a time).
- Faster in raw throughput for short distances, but suffers from **signal skew** (bits arrive at slightly different times) over longer distances.
- Disadvantages: bulky cables, limited length (~2 m max), prone to interference.
- Examples:
  - **LPT Port (Centronics)**: 25-pin D-sub connector, used for printers and scanners.
  - **PATA (Parallel ATA)**: Wide ribbon cable (40/80 wires), used for older hard drives and optical drives.

### Modern Trend — Why Serial Is Dominant Today
- **Higher speeds**: Serial links run at GHz frequencies (USB 3.0 at 5 Gbps, SATA at 6 Gbps).
- **Pin efficiency**: A single serial lane uses fewer pins than a parallel bus, enabling smaller connectors.
- **Differential signaling**: Serial uses twisted-pair wires that cancel electromagnetic interference.
- **Embedded clocking**: Modern serial protocols embed the clock signal in the data stream (8b/10b encoding, NRZ), eliminating clock skew issues.

| Feature | Serial | Parallel |
|---------|--------|----------|
| Data transmission | 1 bit at a time | Multiple bits at once |
| Cable length | Long (up to 5+ m for USB) | Short (~2 m max) |
| Speed potential | Very high (GHz range) | Limited by skew |
| Noise immunity | High (differential) | Lower (single-ended) |
| Connector size | Small | Large |
| Current status | Standard (USB, SATA, PCIe) | Obsolete |

## 8.4 — USB — Versions & Speed

**USB (Universal Serial Bus)** is the most widely used interface for connecting peripherals to a computer. It is hot-swappable, plug-and-play, and provides power to connected devices.

### USB Versions and Speeds

| Version | Also Known As | Speed | Introduced |
|---------|--------------|-------|------------|
| USB 1.0 | Low-Speed | 1.5 Mbps | 1996 |
| USB 1.1 | Full-Speed | 12 Mbps | 1998 |
| USB 2.0 | Hi-Speed | 480 Mbps | 2000 |
| USB 3.0 | SuperSpeed | 5 Gbps | 2008 |
| USB 3.1 Gen 1 | SuperSpeed | 5 Gbps | 2013 |
| USB 3.1 Gen 2 | SuperSpeed+ | 10 Gbps | 2013 |
| USB 3.2 Gen 2×2 | SuperSpeed 20G | 20 Gbps | 2017 |
| USB4 Gen 3×2 | USB4 | 40 Gbps | 2019 |

### USB Connector Types
- **USB Type-A**: Standard rectangular connector (host side).
- **USB Type-B**: Square with beveled corners (printer/scanner side).
- **Micro-USB**: Smaller variant, used for older phones and devices.
- **USB Type-C**: Reversible oval connector (24 pins). Supports:
  - High-speed data (up to USB4 / Thunderbolt 4).
  - **Power Delivery (USB-PD)**: Up to 240 W (48V/5A).
  - **Alternate Modes**: Can carry DisplayPort, HDMI, Thunderbolt signals over the same connector.
- **USB On-The-Go (OTG)**: Allows devices (phones, tablets) to act as USB hosts.

## 8.5 — HDMI, VGA, DisplayPort

These are the primary interfaces used to connect displays (monitors, TVs, projectors) to a computer.

### VGA (Video Graphics Array)
- **Format**: Analog signal — color values are sent as varying voltages.
- **Connector**: 15-pin D-sub (DE-15), 3 rows of 5 pins.
- **Resolution**: Typically up to 2048×1536, but image quality degrades with longer cables due to analog noise.
- **Status**: Legacy — largely replaced by digital interfaces. Still found on older monitors and projectors.
- **No audio** — separate audio cable required.

### HDMI (High-Definition Multimedia Interface)
- **Format**: Digital — transmits uncompressed video and audio over a single cable.
- **Connector**: 19-pin (Type A standard).
- **Versions**: HDMI 1.4 (10.2 Gbps, 4K@30Hz), HDMI 2.0 (18 Gbps, 4K@60Hz), HDMI 2.1 (48 Gbps, 8K@60Hz, 4K@120Hz).
- **Features**: Supports HDCP (copy protection), CEC (device control via remote), ARC/eARC (audio return channel).
- **Common on**: TVs, monitors, game consoles, laptops, streaming devices.

### DisplayPort
- **Format**: Digital — primarily a computer monitor interface.
- **Connector**: 20-pin, with a latching mechanism.
- **Versions**: DisplayPort 1.2 (21.6 Gbps, 4K@60Hz), DP 1.4 (32.4 Gbps, 8K@60Hz), DP 2.0 (80 Gbps, 16K@60Hz).
- **Advantages over HDMI**:
  - Higher bandwidth (especially DP 2.0).
  - **Daisy-chaining** — multiple monitors can be connected in series via MST (Multi-Stream Transport).
  - Free licensing (no royalty fees).
- **Common on**: Computer monitors, high-end graphics cards, professional workstations.

### DVI (Digital Visual Interface)
- **Format**: Supports both digital (DVI-D) and analog (DVI-A), or both (DVI-I).
- **Connector**: 24+5 pins (DVI-I), 24 pins (DVI-D).
- **Resolutions**: Single-link DVI supports up to 1920×1200 @ 60 Hz; dual-link DVI supports up to 2560×1600 @ 60 Hz.
- **Status**: Bridges the VGA-to-HDMI era — still found on older monitors and GPUs. No audio.

| Feature | VGA | DVI | HDMI | DisplayPort |
|---------|-----|-----|------|-------------|
| Signal | Analog | Digital/Analog | Digital | Digital |
| Audio | No | No | Yes | Yes |
| Max Bandwidth | ~400 MHz pixel | ~9.9 Gbps (DL) | 48 Gbps (2.1) | 80 Gbps (2.0) |
| Hot-Plug | No | Yes | Yes | Yes |
| Copy Protection | No | No | HDCP | DPCP |
| Status | Legacy | Legacy | Current | Current |

## 8.6 — PS/2, Ethernet Port, Audio Jack

### PS/2 Port
- **Connector**: Round, 6-pin mini-DIN. Standard color coding: **purple** for keyboard, **green** for mouse.
- **Communication**: Bi-directional serial protocol.
- **History**: Introduced by IBM in 1987 (PS/2 series). Was the standard keyboard/mouse interface through the 1990s and early 2000s.
- **Limitations**: Not hot-swappable (plugging/unplugging while powered on can damage the motherboard).
- **Status**: Legacy — replaced by USB. Still found on some motherboards for legacy compatibility and full n-key rollover support on keyboards.

### Ethernet Port (RJ-45)
- **Connector**: 8P8C modular jack (often called RJ-45).
- **Function**: Wired network connection using Ethernet standards.
- **Speeds**:
  - Fast Ethernet (100BASE-TX): 100 Mbps
  - Gigabit Ethernet (1000BASE-T): 1 Gbps
  - 2.5G/5G BASE-T: 2.5 / 5 Gbps
  - 10 Gigabit Ethernet (10GBASE-T): 10 Gbps
  - Higher speeds (40G, 100G) use fiber optic or specialized copper cables.
- **Cabling**: Typically uses **Cat5e** (up to 1 Gbps) or **Cat6/6a** (up to 10 Gbps) twisted-pair cables.
- **Status**: Widely used — the standard for wired LAN connections.

### Audio Jack (3.5mm TRS/TRRS)
- **Connector**: 3.5 mm (1/8 inch) phone connector.
- **Types**:
  - **TS** (Tip-Sleeve): Mono audio (unbalanced).
  - **TRS** (Tip-Ring-Sleeve): Stereo audio (left + right channels).
  - **TRRS** (Tip-Ring-Ring-Sleeve): Stereo audio + microphone (common on smartphones/laptops).
- **Standard Color Codes on PCs**:
  - **Lime green**: Line out (speakers/headphones).
  - **Pink**: Microphone in.
  - **Blue**: Line in (external audio source).
- **Status**: Still common, but increasingly replaced by USB-C and Bluetooth for audio.
