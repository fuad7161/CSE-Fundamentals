# Input Unit

All devices used to enter data and instructions into a computer system.

### 5.1 — Keyboard — Layout & Types
- Standard QWERTY layout
- Types: mechanical, membrane, ergonomic, virtual/on-screen
- Common keys: alphanumeric, function keys, modifier keys (Ctrl, Alt, Shift), navigation keys

### 5.2 — Pointing Devices
- Mouse (optical, laser, mechanical)
- Trackball, Touchpad, Joystick, Light Pen, Stylus
- Touchscreen (resistive vs capacitive)

### 5.3 — Scanning Devices
- Scanner (flatbed, sheet-fed, handheld)
- OCR (Optical Character Recognition) — converts printed text to digital
- OMR (Optical Mark Recognition) — detects marks on paper (exams/surveys)
- MICR (Magnetic Ink Character Recognition) — used in banking for cheque processing
- Barcode Reader, QR Code Scanner

### 5.4 — Digitizer & Graphics Tablet
- Converts analog drawings/handwriting to digital data
- Used by graphic designers, architects, digital artists

### 5.5 — Voice/Speech Input Devices
- Microphone, headset
- Speech-to-text software, virtual assistants (Siri, Google Assistant)

### 5.6 — Web Camera, Digital Camera
- Webcam: video input for conferencing, streaming
- Digital camera: captures still images and video, transfers to computer
