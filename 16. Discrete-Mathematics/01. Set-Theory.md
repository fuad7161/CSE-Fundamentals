# Set Theory

Foundational concepts of sets — collections of distinct objects — and operations on them.

- **Set Notation** — roster form, set-builder form, universal set, empty set
- **Types of Sets** — finite, infinite, equal, equivalent, subset, power set
- **Set Operations** — union, intersection, difference, symmetric difference, complement
- **Venn Diagrams** — visual representation of set relationships
- **Properties** — commutative, associative, distributive, De Morgan's laws
- **Cardinality** — size of sets, inclusion–exclusion principle
