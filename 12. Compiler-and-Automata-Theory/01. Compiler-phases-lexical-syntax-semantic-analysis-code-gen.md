# Compiler Phases: Lexical, Syntax, Semantic Analysis, and Code Generation

Overview of the major phases of a compiler, from source code to executable output.

- **Lexical analysis:** scanning, tokens, lexemes, and regular expressions
- **Syntax analysis (parsing):** context-free grammars, parse trees, AST construction
- **Semantic analysis:** type checking, scope resolution, symbol table management
- **Intermediate code generation:** three-address code, SSA form
- **Optimization:** constant folding, dead code elimination, loop optimization
- **Code generation:** register allocation, instruction selection, target code output
- Symbol table and error handling throughout the phases
