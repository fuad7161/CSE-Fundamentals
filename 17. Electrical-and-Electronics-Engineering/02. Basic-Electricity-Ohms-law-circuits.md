# Basic Electricity — Ohm's Law, Circuits

Core principles of electricity and basic circuit analysis methods.

- **Fundamental Quantities** — voltage (V), current (I), resistance (R), power (P)
- **Ohm's Law** — V = IR, power dissipation P = VI = I²R
- **Circuit Components** — resistors, capacitors, inductors, sources (DC/AC)
- **Series Circuits** — equivalent resistance, voltage divider rule
- **Parallel Circuits** — equivalent conductance, current divider rule
- **Kirchhoff's Laws** — KCL (current law), KVL (voltage law)
- **DC Circuit Analysis** — mesh analysis, nodal analysis
- **AC Circuits** — impedance, phase, sinusoidal steady-state analysis
- **Measuring Instruments** — ammeter, voltmeter, ohmmeter, multimeter
